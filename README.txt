Add a deployment button to trigger static site rebuilds via webhook.

== Description ==
Site Deployment Manager adds a deployment button to your WordPress admin panel that triggers a webhook to rebuild your static site. It includes logging functionality to track deployment history.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/site-deployment-manager`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your webhook URL in the plugin settings
